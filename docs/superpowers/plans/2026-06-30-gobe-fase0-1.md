# Gobe Fase 0 + Fase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build Gobe, an Android TV app that scans real ROM folders on the ONN Plus 4K and presents a D-pad-navigable library grouped by system — no emulation yet.

**Architecture:** Single Kotlin/Gradle app module. Compose for TV UI (`androidx.tv`), Room persistence, Coroutines/Flow, minimal manual DI. Pure logic (system detection, name cleaning, scan diff) is unit-tested first; Room/UI are verified with light instrumented + on-device checks. `MANAGE_EXTERNAL_STORAGE` gives direct `java.io.File` access (no SAF, no bundled ROMs).

**Tech Stack:** Kotlin, Jetpack Compose for TV (`androidx.tv:tv-material3`, `androidx.tv:tv-foundation`), Room, Kotlin Coroutines/Flow, AndroidX Lifecycle/Activity-Compose, Gradle Kotlin DSL with wrapper, JDK 17, Android SDK command-line tools, `adb`.

**Spec:** [docs/superpowers/specs/2026-06-30-gobe-fase0-1-design.md](../specs/2026-06-30-gobe-fase0-1-design.md)

## Task 1: Connect to the ONN and confirm Android version + ROM path

Environment setup. Goal: `adb` reaches the ONN; we learn the real `minSdk`-relevant version and the actual ROM folder path to use as the default.

**Files:** none yet (findings feed Task 2 + Task 8 default path).

- [ ] **Step 1: Enable wireless debugging on the ONN**

On the device: Settings → System → About → click "Android TV OS build" 7× to unlock Developer options → Settings → System → Developer options → enable "Wireless debugging" (and "USB debugging"). Note the IP:port shown under Wireless debugging.

- [ ] **Step 2: Pair + connect from the Mac**

Run (use the pairing code + port the device shows under "Pair device with pairing code"):
```bash
adb pair <ONN-ip>:<pair-port>     # enter the 6-digit code when prompted
adb connect <ONN-ip>:<debug-port>
adb devices
```
Expected: the device appears as `device` (not `unauthorized`/`offline`).

- [ ] **Step 3: Confirm Android version → finalize minSdk**

Run:
```bash
adb shell getprop ro.build.version.release   # e.g. 12
adb shell getprop ro.build.version.sdk       # e.g. 31
```
Expected: SDK ≥ 30. Record it. minSdk stays 30 unless the device is higher and we want to bump; keep 30 to be safe.

- [ ] **Step 4: Find the real ROM folder path**

Run:
```bash
adb shell ls /storage/emulated/0
adb shell 'ls -R /storage/emulated/0 2>/dev/null | grep -iE "rom|nes|snes|n64|emu" | head'
```
Inspect output to locate where ROMs already live (the user copied them via FTP). Record the exact path (e.g. `/storage/emulated/0/Roms`). This becomes `DEFAULT_ROM_PATH` in Task 8.

- [ ] **Step 5: Verify a known ROM file is readable**

Run: `adb shell ls -la "<found-rom-path>"`
Expected: lists `.nes`/`.smc`/`.z64`/`.zip` files. If empty, ask the user where they put the ROMs before continuing.

```kotlin
package com.gobe.tv.data.scan

import com.gobe.tv.data.system.NameCleaner
import com.gobe.tv.data.system.SystemDetector
import com.gobe.tv.domain.System
import java.io.File

data class ScannedRom(
    val path: String,
    val system: System,
    val displayName: String,
    val fileName: String,
    val sizeBytes: Long,
)

class RomScanner(
    private val detector: SystemDetector,
    private val nameCleaner: NameCleaner,
    private val maxDepth: Int = 12,
) {
    fun scan(folderPaths: List<String>): List<ScannedRom> {
        val out = mutableListOf<ScannedRom>()
        for (p in folderPaths) {
            val root = File(p)
            if (!root.isDirectory) continue
            walk(root, 0, out)
        }
        return out
    }

    private fun walk(dir: File, depth: Int, out: MutableList<ScannedRom>) {
        if (depth > maxDepth) return
        val children = dir.listFiles() ?: return
        for (f in children) {
            // Do not follow symlinks (avoid cyclic/pathological trees).
            if (isSymlink(f)) continue
            if (f.isDirectory) {
                walk(f, depth + 1, out)
            } else {
                val system = detector.detect(f.absolutePath) ?: continue
                out += ScannedRom(
                    path = f.absolutePath,
                    system = system,
                    displayName = nameCleaner.clean(f.name),
                    fileName = f.name,
                    sizeBytes = f.length(),
                )
            }
        }
    }

    private fun isSymlink(f: File): Boolean =
        try { f.canonicalFile != f.absoluteFile } catch (e: Exception) { true }
}
```

```kotlin
package com.gobe.tv.data

import com.gobe.tv.data.scan.ScannedRom

data class LibraryDiffResult(
    val toInsert: List<ScannedRom>,
    val toDeletePaths: List<String>,
)

object LibraryDiff {
    fun compute(scanned: List<ScannedRom>, existingPaths: Set<String>): LibraryDiffResult {
        val scannedPaths = scanned.map { it.path }.toSet()
        val toInsert = scanned.filter { it.path !in existingPaths }
        val toDelete = existingPaths.filter { it !in scannedPaths }
        return LibraryDiffResult(toInsert, toDelete)
    }
}
```

```kotlin
package com.gobe.tv.data

import com.gobe.tv.data.db.GameDao
import com.gobe.tv.data.db.GameEntity
import com.gobe.tv.data.db.RomFolderDao
import com.gobe.tv.data.db.RomFolderEntity
import com.gobe.tv.data.scan.RomScanner
import com.gobe.tv.domain.Game
import com.gobe.tv.domain.RomFolder
import com.gobe.tv.domain.System
import kotlinx.coroutines.flow.Flow
import kotlinx.coroutines.flow.map

class LibraryRepository(
    private val gameDao: GameDao,
    private val folderDao: RomFolderDao,
    private val scanner: RomScanner,
    private val now: () -> Long = { java.lang.System.currentTimeMillis() },
) {
    fun observeGames(): Flow<List<Game>> = gameDao.observeAll().map { list ->
        list.map { it.toDomain() }
    }

    fun observeFolders(): Flow<List<RomFolder>> = folderDao.observeAll().map { list ->
        list.map { RomFolder(it.id, it.path, it.enabled) }
    }

    suspend fun ensureDefaultFolder(defaultPath: String) {
        if (folderDao.count() == 0) folderDao.insert(RomFolderEntity(path = defaultPath))
    }

    suspend fun addFolder(path: String) = folderDao.insert(RomFolderEntity(path = path))
    suspend fun setFolderEnabled(id: Long, enabled: Boolean) = folderDao.setEnabled(id, enabled)
    suspend fun removeFolder(id: Long) = folderDao.delete(id)

    suspend fun getGame(id: Long): Game? = gameDao.getById(id)?.toDomain()

    /** Scans enabled folders and reconciles the DB. Returns number of games after scan. */
    suspend fun rescan(): Int {
        val paths = folderDao.getEnabled().map { it.path }
        val scanned = scanner.scan(paths)
        val existing = gameDao.getAll().map { it.path }.toSet()
        val diff = LibraryDiff.compute(scanned, existing)
        if (diff.toDeletePaths.isNotEmpty()) gameDao.deleteByPaths(diff.toDeletePaths)
        if (diff.toInsert.isNotEmpty()) {
            val ts = now()
            gameDao.insertAll(diff.toInsert.map {
                GameEntity(
                    path = it.path, system = it.system, displayName = it.displayName,
                    fileName = it.fileName, sizeBytes = it.sizeBytes, dateAdded = ts,
                )
            })
        }
        return gameDao.getAll().size
    }

    private fun GameEntity.toDomain() = Game(
        id, path, system, displayName, fileName, sizeBytes, lastPlayed, dateAdded
    )
}
```

```kotlin
val lifecycleOwner = androidx.lifecycle.compose.LocalLifecycleOwner.current
DisposableEffect(lifecycleOwner) {

        if (event == androidx.lifecycle.Lifecycle.Event.ON_RESUME) {
```
